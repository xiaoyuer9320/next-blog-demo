# Next.js 全栈技术博客 (Fullstack Blog)

基于 Next.js 15 (App Router) + Supabase + Tailwind CSS 构建的全栈内容管理系统。

## 🚀 在线预览
[点击访问在线 Demo](https://你的vercel域名.vercel.app)
*(注：由于部署在 Vercel，国内访问可能需要网络环境支持)*

## ✨ 核心功能
- **全栈开发**：使用 Server Actions 处理后端逻辑，无需独立 API。
- **完整 CRUD**：支持文章的发布、浏览、编辑与删除。
- **数据持久化**：集成 Supabase (PostgreSQL) 云数据库。
- **极致体验**：实现 SSR 服务端渲染，集成 Loading 骨架屏与 Markdown 渲染。
- **SEO 优化**：动态生成 Metadata，支持搜索引擎收录。

## 🛠️ 技术栈
- **Framework**: Next.js 15, React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase
- **Deployment**: Vercel
